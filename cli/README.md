# @axon-xlayer/start

Quick-start CLI for AXON — Neural Intelligence Layer for X Layer.

## Usage

```bash
npx @axon-xlayer/start
```

For AI agents (machine-readable JSON):
```bash
npx @axon-xlayer/start --json
```

## What is AXON?

AXON is a production MCP skill server exposing 45 tools for X Layer. Any AI agent can call tools with a single HTTP request — no account, no API key.

- **Free tools**: gas price, token security, smart money signals, on-chain oracle reads
- **Premium tools**: wallet analysis, arbitrage scanner (x402 OKB micropayments)
- **On-chain oracle**: AxonVerdictLedger stores every security verdict permanently on X Layer
